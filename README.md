# 📊 2023 Sales Performance Dashboard (Excel Pivot Table Project)

## 📌 Project Overview

This project analyzes **2023 company sales performance** using **Microsoft Excel Pivot Tables**.
The goal was to transform raw transactional data into actionable business insights through executive dashboards, KPI summaries, trend analysis, and profitability breakdowns.

This project demonstrates practical skills in:

* Pivot Tables
* Data Cleaning
* KPI Reporting
* Executive Dashboards
* Trend Analysis
* Profitability Analysis
* Interactive Filters / Slicers
* Business Decision Making

---

## 📈 Key Business Insights

### 1️⃣ Executive KPI Dashboard

| Metric            | Value      |
| ----------------- | ---------- |
| Total Sales       | $2,059,070 |
| Total Profit      | $2,250     |
| Total Orders      | 1,000      |
| Avg Profit Margin | 0.99%      |

📌 Insight: High revenue but extremely low margin indicates pricing or cost control issues.

---

### 2️⃣ Regional Performance

| Region  | Sales    | Profit  |
| ------- | -------- | ------- |
| East    | $444,567 | $4,741  |
| North   | $441,448 | $7,126  |
| West    | $433,278 | -$3,779 |
| South   | $383,718 | -$1,131 |
| Central | $356,060 | -$4,707 |

📌 Insight: North and East are profitable. West and Central need operational review.

---

### 3️⃣ Customer Segment Analysis

| Segment        | Sales    | Profit  |
| -------------- | -------- | ------- |
| Consumer       | $543,500 | -$3,150 |
| Small Business | $529,024 | $5,344  |
| Home Office    | $512,164 | -$5,100 |
| Corporate      | $474,382 | $5,157  |

📌 Insight: Small Business and Corporate are strongest profit drivers.

---

### 4️⃣ Product Category Performance

| Category        | Sales    | Profit  |
| --------------- | -------- | ------- |
| Clothing        | $548,390 | -$4,230 |
| Electronics     | $511,126 | -$2,206 |
| Office Supplies | $505,941 | $4,650  |
| Furniture       | $493,614 | $4,036  |

📌 Insight: Clothing and Electronics require pricing strategy changes.

---

### 5️⃣ Monthly Trend Analysis

* Strongest Month: **December**
* Weakest Month: **February**
* Best Quarter: **Q4**

📌 Insight: Year-end seasonality strongly impacts revenue.

---

### 6️⃣ Shipping Efficiency

| Mode           | Result          |
| -------------- | --------------- |
| Standard Class | Highest Profit  |
| Same Day       | Negative Profit |

📌 Insight: Faster delivery increased volume but reduced margin.
